# Weyland Observatory

This project exists to understand how cities operate.

It is not currently a startup.

It is not currently a product.

It is a research environment.

Every notebook should answer a question.

Every visualization should help generate better questions.

Every hypothesis should eventually be tested in the real world through field observations or stakeholder interviews.

Software exists to support investigation, not the other way around.

The intended workflow is:

Question
→ Data
→ Analysis
→ Visualization
→ Hypothesis
→ Field Investigation
→ Stakeholder Conversation
→ New Question

The goal is to develop a deep operational understanding of urban maintenance systems before attempting to build solutions.
